# インストールガイド

このChrome拡張機能をインストールする手順です。

## ステップ1: アイコンの生成（オプション）

アイコンがないと警告が表示されますが、拡張機能は動作します。
アイコンを生成する場合は以下のいずれかを実行してください：

### Linuxの場合

```bash
./setup.sh
```

### 手動でアイコンを作成する場合

`icons/`フォルダに以下のファイルを配置：
- `icon16.png` (16x16px)
- `icon48.png` (48x48px)
- `icon128.png` (128x128px)

詳細は[icons/README.md](icons/README.md)を参照してください。

## ステップ2: Chrome拡張機能として読み込む

1. **Chromeを開く**

   Google ChromeまたはChromiumベースのブラウザを起動します。

2. **拡張機能ページを開く**

   アドレスバーに以下を入力してEnter:
   ```
   chrome://extensions/
   ```

3. **デベロッパーモードをON**

   ページ右上の「デベロッパーモード」トグルをONにします。

4. **拡張機能を読み込む**

   「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。

5. **フォルダを選択**

   このプロジェクトのフォルダ（`breakLoopTimer`）を選択します。

6. **完了！**

   拡張機能がインストールされました。アイコンがツールバーに表示されます。

## ステップ3: 初期設定

1. **拡張機能アイコンをクリック**

   ツールバーの拡張機能アイコンをクリックします。

2. **設定を開く**

   ポップアップから「⚙️ 設定」ボタンをクリックします。

3. **お好みに合わせて設定**

   - 休憩時間（デフォルト: 10分）
   - 警告時間（デフォルト: 1分前）
   - リダイレクト先URL（デフォルト: Notion）
   - 通知の有効/無効

## 使用開始

TwitterまたはX（旧Twitter）を開くと、自動的にバナーが表示されます！

## トラブルシューティング

### 拡張機能が読み込めない

- すべてのファイルが正しくダウンロードされているか確認してください
- `manifest.json`が存在するか確認してください

### バナーが表示されない

- Twitter/Xのページをリロードしてみてください
- `chrome://extensions/` で拡張機能が有効になっているか確認してください
- デベロッパーツールのConsoleでエラーがないか確認してください

### 通知が表示されない

- ブラウザの通知権限を確認してください
- 拡張機能の設定で「デスクトップ通知を有効にする」がONになっているか確認してください
- システムの通知設定でChromeの通知が許可されているか確認してください

### タイマーが動作しない

- バックグラウンドページのコンソールでエラーを確認：
  1. `chrome://extensions/` を開く
  2. 拡張機能の「詳細」をクリック
  3. 「service worker」のリンクをクリック
  4. デベロッパーツールでエラーを確認

## アップデート方法

1. `chrome://extensions/` を開く
2. 拡張機能の更新ボタン（🔄）をクリック
3. ページをリロード
